📌 Project Overview

The Empathetic Cognitive Support Agent (ECSA) is an AI-powered customer support chatbot designed to provide emotionally intelligent and context-aware responses. Unlike traditional rule-based chatbots, ECSA combines Natural Language Processing (NLP), Transformer-based semantic intent detection, and sentiment analysis to understand user queries, detect emotions, and generate adaptive responses. The chatbot also incorporates confidence-based escalation to a human support agent whenever it is uncertain, ensuring reliable and user-centric assistance.

🎯 Project Objectives
Build an AI-powered customer support chatbot.
Detect user intent using transformer embeddings.
Analyze user emotions through sentiment analysis.
Generate adaptive and empathetic responses.
Escalate low-confidence queries to human support.
Simulate natural human-like conversations.

⚙️ AI Workflow
Receive user query.
Detect user emotion using TextBlob sentiment analysis.
Convert query into transformer embeddings.
Match query with predefined intents using cosine similarity.
Select an emotion-aware response.
Calculate confidence score.
Escalate to a human agent if confidence falls below the threshold.
Display the chatbot response with detected emotion and confidence score.
